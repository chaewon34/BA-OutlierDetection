# BA-OutlierDetection

## HOW TO CONTIRIBUTE

1. fork respository
2. git clone

   `git clone https://github.com/<user-id>/BA-OutlierDetection.git`

3. 작업 branch 생성

   `git checkout -b "<branch name>"`

## HOW TO RUN

1. cd client
2. build frontend

   `npm run build`

3. cd ..
4. docker compose

   `docker-compose up --build`
